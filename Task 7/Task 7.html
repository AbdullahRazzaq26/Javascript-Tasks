Task 7: Weather Advisory

Description: Ek program banaiye jo temperature ke hisaab se weather advisory de:

Instructions:

  1. Agar temperature 100°F se zyada hai, toh "Extreme Heat Advisory"
  2. Agar temperature 80°F se 99°F hai, toh "Warm Weather Advisory"
  3. Agar temperature 60°F se 79°F hai, toh "Pleasant Weather"
  4. Agar temperature 32°F se 59°F hai, toh "Cool Weather"
  5. Agar temperature 32°F se kam hai, toh "Freezing Temperature Advisory"
